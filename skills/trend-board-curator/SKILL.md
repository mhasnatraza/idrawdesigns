# Trend Board Curator

## Workflow
1. Collect 10–30 recent references around one niche/topic.
2. Cluster into 3–5 themes (visual, messaging, motion, format).
3. Label each theme with audience relevance and funnel fit.
4. Produce practical outputs:
   - moodboard summary
   - do/do-not style cues
   - campaign concept ideas
5. Recommend one low-risk trend to test first.

## Output format
- Theme name
- Evidence references
- Why it is rising
- Brand fit score (1–5)
- Test concept
- Production complexity (low/medium/high)
